StingriX — an experimental kernel

A personal kernel project built and maintained by a single teenage developer. This site documents goals, progress, and how to follow along.

⚠️ WARNING: StingriX is an experimental kernel made by a single teenage developer. There is no stable public release yet. Use only in safe testing environments.

What is StingriX?

StingriX is a small, learning-first kernel built from scratch. It is not Linux-based; it explores kernel design and the boot process for minimal systems. The initial public release will be an extremely minimal beta for testing core functionality.

Initial goals

  • Boot a tiny, auditable kernel that demonstrates core concepts.
  • Serve as a learning platform for low-level development.
  • Grow gradually — features and hardware support will be added over time.

Expectations

  • The first public beta will be minimal and mostly for testing.
  • Progress is iterative and paced by a single developer.
  • Feedback and constructive testing help the project improve.
