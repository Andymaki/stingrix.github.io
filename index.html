StingriX — an experimental kernel

A personal kernel project built and maintained by a single teenage developer. This website documents the project's goals, progress and how to follow along.

⚠️ WARNING: StingriX is an experimental kernel made by a single teenage developer. There is no stable public release yet. Installing pre-release software can cause data loss, hardware issues or other problems. Use only in safe testing environments.

What is StingriX?

StingriX is a small, learning-first kernel built from scratch. It is not Linux-based: it is intended to explore kernel design, low-level programming and the steps required to boot and run minimal systems. The initial public release will be an extremely minimal beta for testing core functionality.

Initial goals

  • Provide a tiny, auditable kernel that boots and demonstrates core concepts.
  • Serve as a learning platform for low-level development.
  • Grow gradually — features and hardware support will be added over time.

Expectations

  • The first public beta will be minimal and mostly for testing.
  • Because this is a one-person project, progress will be iterative and paced by the developer.
  • Contributions, feedback and constructive testing help the project improve.
